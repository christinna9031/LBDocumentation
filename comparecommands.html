Compare Commands

Compare: Skip if

Prevents the following commands from happening if the math is true.
Remember to always compare the same types of values - either strings or numbers.
You can use nested Compare: Skip if commands to check for multiple conditions or to make the execution more effective.
Make sure the delay for Compare: Skip if command is always equal or smaller than the delay for the following commands you want to include in your skip if steps. Otherwise the commands will get executed regardless of the math result.

Name Type Description
Variable String Variable name made from Math: Change Variable command
Math Dropdown menu Name of the string to parse. Use /$variable$/ if you used File: Load String command.
Value String/Real Name of the string to parse. Use /$variable$/ if you used File: Load String command.
Step Int Number of commands to skip if the result is true.

Compare: End if

Prevents ALL commands from happening (and ends them prematurely) if the math is true. Does not affect other buttons.
Remember to always compare the same types of values - either strings or numbers.
You can use this command to temporarily disable a button.

Name Type Description
Variable String Variable name made from Math: Change Variable command
Math Dropdown menu Math to execute and check if it's true or false.
Value String/Real Value to compare to the variable value.

Compare: Redo if

Reactivates the following commands for as long as the math is true. Once it reactivates the commands, delay does not matter, as it will make the commands happen on a frame per second basis.
Remember to always compare the same types of values - either strings or numbers.
This is a very useful command to:

  • Continuously check whether a value has changed and immediately react to it. For example, you can make a button change its color based on which scene you're currently on.

  • Make a simple timer using Variable: Transition command.

  • Make a smooth filter transition from value A to value B. For example, you can make a smooth crop animation.

Name Type Description
Variable String Variable made from Math: Change Variable command
Math Dropdown menu Math to execute and check if it's true or false.
Value String/Real Value to compare to the variable value.
Step Int Number of commands to redo if the result is true.
Self boolean Whether you want to include the command itself in the loop.
If set to false, it will repeat only once.

Compare: Redo Previous if

Same as Compare: Redo if, but reactivates the previous commands instead.
Remember to always compare the same types of values - either strings or numbers.

Name Type Description
Variable String Variable made from Math: Change Variable command.
Math Dropdown menu Math to execute and check if it's true or false.
Value String/Real Second variable to compare to the first one.
Step Int Number of commands to redo if the result is true.
Self boolean Whether you want to include the command itself in the loop.
If set to false, it will repeat only once.
