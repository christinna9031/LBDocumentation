Known Bugs

When I try to link my Twitch account, it says this site can't be reached.
This has been fixed. Please download the newest version.

When I click on -Help- in LB commands, the page doesn't work!
Commands.html is saved locally in your LioranBoard folder, you can access it from there instead. The most recent IE/Chrome update causes this link to be malformed.

When I use the command Open URL, it doesn't work.
This has been fixed. Please download the newest version.

Every time I close OBS, it crashes.
This has nothing to do with LB. It is a known bug that happens with some plugins (like obs-websocket) installed in OBS.

I get a grey window with an error message when I try to install X extension.
That sometimes happens due to extra spaces in the file. Press ignore and the extension should install just fine.

Wild cards don't work for Point Redeem name.
This has been fixed. Please download the newest version.

Multiple accounts do not work when refreshing Transmitter
When using multiple accounts, the main account does not reconnect when Transmitter is refreshed, and will not listen to any alerts. Restart LB Receiver to reconnect to all your accounts.

Some special symbols and foreign language letters don't show up in the input fields
This is just a visual bug, the symbols are saved just fine (you can open the deck .ini file and check).

Cannot set up a macro key to “9”. It says unknown.
This is just a visual text bug. The macro still works and presses the key.
