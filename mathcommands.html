Math Commands

Math: Change Variable

Create or change a variable. If you try to execute math (such as +=) on a variable that was not created, it will be set to 0.

Variables are containers for storing data values (including numbers and strings), that can be reused over and over again. Variable names are case sensitive, meaning that "Variable" and "variable" are not going to have the same value.

All variables inside Receiver are global and get deleted every time you close or reset Receiver. They are not unique to a button that created them and any button/deck can access them.
You can press Tab in the main screen of the Receiver to view all the existing variables, strings will be wrapped in " ".
Variables can be used as parameters or inside any string. In this case you need to wrap them like this: /$variable$/. Wrapping variables tells LioranBoard to work with the value of the variable rather than the variable itself.



Pay special attention to the operators. = is NOT the same as =string. For example, if you try to execute a string addition on a real value (number), it will result in an error and Receiver crashing.


Name Type Description
Variable String Name of the variable. Accepts anything including spaces and numbers.
Math Dropdown menu Type of math to execute on the variable
Value Real/String Number/String to process. Must be a direct value, not a variable name. In this case use Math: Variable to Variable command.

Math: Variable to Variable

Same as Math: Change Variable command, but processes two different variables rather than variable + direct value. Refer to Math: Change Variable for more info.

Name Type Description
Variable String Name of the variable. Accepts anything including spaces and numbers.
Math Dropdown menu Type of math to execute on the variables.
Variable String Name of the variable to process. Must be already created.

Math: Variable Transition

Transition a variable from a starting value to a final value over a given duration.
This command can act as a timer to trigger other commands or buttons.
For example, if you set the start value to 10, final value to 0 and duration to 10000ms (=10 seconds), it will decrease the value by 1 every second and reach 0 after exactly 10 seconds.

Name Type Description
Variable String Name of the variable. Accepts anything including spaces and numbers.
Start Value Real Starting point (where the counting starts from)
Final Value String Finishing point (where the counting stops at)
Allow Float boolean Float=values with decimal points, int=values without decimal points (rounded down)
Dur.(MS) Int Duration of the transition

Math: Clamp

Clamps a variable (number) between two values..

  1. If value of the variable < minimum allowed value, returns minimum allowed value.
  2. If value of the variable > maximum allowed value, returns maximum allowed value.
  3. If value of the variable is within the allowed range, returns the original value.

Do not use Math:Clamp on strings and String:Clamp on real values, as this will result in an error and LB crashing.

Name Type Description
Variable String Name of a variable to clamp
Min Real Minimum value allowed
Max Real Maximum value allowed

Math: Random

Returns a variable with a random value (int) between minimum and maximum possible value.
This is a basic command you can use for showing random images, playing random videos/sounds, picking a random viewer from chat etc.
If you wish to have more power over randomizing values (like adding weights), it is best to use Stacks instead.

For example, if you have 10 different images in a folder with the same name formatting (image1.png, image2.png, image3.png) and you wish to pick a random image and trigger it in OBS, you can use Math:Random and Source Change Settings command.

Name Type Description
Variable String Name of a variable to save the random value
Min Real Minimum value possible
Max Real Maximum value possible
Float boolean If Float (number with decimal points) is set to True, it will return a random value between maximum and minimum possible value, but never the maximum or minimum value itself.
If Float is set to False, it will return a random value, including the maximum and minimum possible value.
Any float value in minimum or maximum value will be omitted if set to False.

Math: Clear Variable

Deletes a variable. It will also remove it from the list of variables (when you press Tab in the main Receiver menu).
Useful command to keep your list clean and uncluttered.

Name Type Description
Variable String Name of a variable to delete

Math: String to Real

Turns a string into a real value (=number).

  • Turning a string "-1.20" into a real value returns -1.20.
  • Turning a string "Hello World" into a real value returns 0.
  • Turning a string "123 Hello World" into a real value returns 123.
  • Turning a string "Hello World 123" into a real value returns 0.
  • Turning a string "123 Hello World 456" into a real value returns 123.
Name Type Description
Variable String Name of a variable to save the string as a real value
String Variable String Name of a variable containing the string you wish to become real

Math: Hex to Real

Converts a hexadecimal string into a real (decimal) number.

Name Type Description
Variable String Name of a variable to save the hex as a real value
String Variable String Name of a variable containing the hex you wish to become real

Math: Pull Source Value

Pulls the last known value of a source. Will return 0 if the source doesn't exist.
This may not be accurate if the source is inside a group. In this case try putting the group name instead of the scene name or using a nested scene instead of a group.

Very useful command to use together with Source:Motion commands. For example, you can pull the current X and Y position of a source and then use Source Motion: Position to smoothly move it across the screen.

Name Type Description
Scene String Scene name containing the source
Source String Source name to pull the value
Variable String Variable name to save the pulled value into
Pull Dropdown menu
  • X/Y position - returns position of the source
  • X/Y scale - returns scale of the source, 1=100%
  • Rotation - returns the degree of rotation of the source (0-360)
  • Volume - returns the volume percentage of the source (0-100)
  • Muted - returns 1 or 0 whether the source is muted or not
  • Visible - returns 1 or 0 whether the source is visible or not

Math: Pull Filter Visibility

Pulls the last known visibility of a specified filter

Name Type Description
Source String Source name containing the filter
Filter String Name of a filter to pull visibility from
Variable String Variable name to save the pulled value into
